blank：空白页面

buttons：按钮、icon

icon两种类名调用方式：menu-icon fa fa-tachometer（其中menu-icon是必备类后面两个根据icon的样式不同选取其他类名）和ace-icon fa fa-gift（其中ace-icon是必备类后面两个根据icon的样式不同选取其他类名）

calendar：日历标记备忘录

content-slider：模态框内容滑块

dropzone：多张图片上传

elements：用户界面元素，进度条、手风琴效果、tips提示、下拉菜单、

email、email-confirmation、email-contrast、email-navbar、email-newsletter：邮件模板

form-elements-1、form-elements-2：表单元素

form-wizard：表单导向与验证

gallery：图片画廊可点击放大

grid：栅格化布局

inbox：邮箱，收发送邮件

invoice：客户发票列表

jqgrid：固定高度滑动来显示的表格

jquery-ui：Datepicker日期控件,Dialogs对话框,Autocomplete自动补全输入，Menu菜单， Spinner加减，Slider进度滑块，Sortable

Accordion可排序的手风琴排版，Tabs切换，Progressbar进度条，Selectmenu下拉选择框

login：登录页面

mobile-menu-1：页面宽度小于992px时，左侧导航图标悬浮左上角

mobile-menu-2：页面宽度小于992px时，左侧导航变成细条状

mobile-menu-3：页面宽度小于992px时，左侧导航图标悬浮右上角

nestable-list：嵌套的列表

pricing：商品价位

profile：用户信息页面

search：搜索

tables：表格

timeline：时光轴

top-menu：导航置顶

treeview：树形图

two-menu-1：左侧导航和顶部导航，即双导航展示

two-menu-2：左侧导航和顶部导航，即双导航展示

typography：页面排版

widgets：窗口小部件，可以拖拽进行位置移动

wysiwyg：编辑器
